HTML:Ol

The li element represents a list item. If its parent element is an ol, or ul, then the element is an item of the parent element's list, as defined for those elements. Otherwise, the list item has no defined list-related relationship to any other li element. If the parent element is an ol element, then the li element has an ordinal value.
