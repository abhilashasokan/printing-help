HTML:Div

The DIV elements, in conjunction with the id and class attributes, offer a generic mechanism for adding structure to documents. These elements define content to be block-level but impose no other presentational idioms on the content. Thus, authors may use these elements in conjunction with style sheets, the lang attribute, etc., to tailor HTML to their own needs and tastes.

Suppose, for example, that we wanted to generate an HTML document based on a client information. 

<!-- Example of data from the client information: -->
<!-- Name: Stephane Boyera, Tel: (212) 555-1212, Email: sb@foo.org -->
<div id="client-boyera" class="client">
    <table class="client-data">
        <tr>
            <th>Last name:</th>
            <td>Boyera</td>
        </tr>
        <tr>
            <th>First name:</th>
            <td>Stephane</td>
        </tr>
        <tr>
            <th>Tel:</th>
            <td>(212) 555-1212</td>
        </tr>
        <tr>
            <th>Email:</th>
            <td>sb@foo.org</td>
        </tr>
    </table>
</div>
